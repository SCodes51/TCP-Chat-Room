TCP Chat Room
==============

Desciption : Chat Room consists One Server with Multiple Clients. Clients can connect to the server which works as a chatroom and each client can send messages to the server with their nickname and all the other clients can see these messages and respond.


Open CMD's,  one for server and as many for clients:
---------------------------------------------------
--> Server command :   python server.py
--> Client's commmand :   python client.py
